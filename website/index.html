功能特性

完整的电台操控体验

从频谱瀑布到 DSP 降噪，从频道记忆到 PTT 发射 —— 所有功能都经过精心设计，专为 iPhone 触摸交互优化。

📊

实时频谱瀑布

512-bin FFT 实时频谱，与 Web 前端匹配的颜色渐变。自适应噪声基底，可配置增益/偏置对比度拉伸。

  • 动态频率刻度（39k/78k/156k/312k IQ 带宽）
  • 点击瀑布图直接跳频
  • 5 帧累积 + 10fps 节流，零主线程开销
🎧

低延迟音频

RX：Int16 PCM 48kHz 通过 AVAudioPlayerNode 播放。TX：麦克风采集 → 降采样 → Int16 PCM → WebSocket。

  • RX 48kHz 高清音频回放
  • TX 16kHz 单声道采集
  • 实时 RMS 电平表
  • 一键静音
🎛️

频率 & 模式控制

超大 56pt 等宽频率显示，步进调谐，12 个业余波段预设，USB/LSB/CW/AM/FM/WFM 六种模式。

  • 56pt 数字频率显示，点击输入
  • 5 档步进：1K/5K/10K/50K/100K
  • 160m–2m 共 12 个波段预设
  • CW/SSB/Wide/AM/FM 滤波器预设
🔊

DSP 降噪面板

完整的 WDSP 数字信号处理控制：NR2 降噪、NB 噪声消除、ANF 自动陷波、AGC 自动增益。

  • NR2 降噪 + 电平调节
  • NB / ANF / NF 独立开关
  • AGC：关 / 慢 / 中 / 快
  • 手动陷波列表（添加/删除）
💾

频道记忆

3×3 网格频率预设，始终可见。一键切换频率和模式。空槽位显示占位符，随时快速保存。

  • 9 个频道槽位，3×3 网格布局
  • UserDefaults JSON 持久化
  • 滑动删除，快速保存
  • 点击即调谐频率+模式
⚙️

系统设置

服务器地址配置、IQ 采样率选择（39k/78k/156k/312k）、Keychain 自动登录、连接状态诊断。

  • 4 档 IQ 采样率可切换
  • Keychain 安全存储密码
  • 4 路 WebSocket 连接状态指示
  • 收藏频道管理

快速开始

5 分钟上手

克隆、生成 Xcode 项目、构建、连接电台 —— 只需四步。

克隆仓库

从 GitHub 获取最新代码

git clone https://github.com/cheenle/SunsdrMobile.git

生成 Xcode 项目

使用 XcodeGen 从 project.yml 生成 .xcodeproj

cd SunsdrMobile && xcodegen generate

构建 & 运行

在 Xcode 中打开项目，选择 iPhone 真机，Cmd+R 运行

open SunsdrMobile.xcodeproj

连接电台

输入服务器地址和密码，自动登录并建立 4 路 WebSocket 连接

radio.vlsc.net:8889 → Keychain
在 GitHub 上查看
